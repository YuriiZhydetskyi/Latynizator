Політика конфіденційності для розширення "Латинізатор"

Останнє оновлення: 01.08.2024

1. Вступ

"Латинізатор" зобов'язується захищати вашу конфіденційність. Ця Політика конфіденційності пояснює, як наше розширення збирає, використовує та захищає вашу інформацію при використанні нашого розширення для Chrome.

2. Збір та використання інформації

Наше розширення збирає та використовує наступну інформацію:

  • Налаштування користувача: Ми зберігаємо ваші мовні налаштування та налаштування активації (глобальні, для конкретних сайтів та сторінок) локально у вашому браузері, використовуючи API сховища Chrome. Ця інформація використовується виключно для забезпечення послідовного користувацького досвіду і не передається на наші сервери або будь-яким третім сторонам.
  • Текст для латинізації: Коли ви використовуєте наше розширення для латинізації тексту, вибраний текст обробляється локально у вашому браузері. Ми не збираємо, не зберігаємо і не передаємо вміст, який ви латинізуєте.

3. Зберігання даних

Всі дані, зібрані "Латинізатором", зберігаються локально на вашому пристрої за допомогою API сховища Chrome. Ми не використовуємо жодних зовнішніх серверів або баз даних для зберігання вашої інформації.

4. Поширення даних

"Латинізатор" не поширює жодних користувацьких даних третім сторонам. Всі операції виконуються локально у вашому браузері.

5. Безпека

Ми зобов'язуємось забезпечити безпеку вашої інформації. Оскільки всі дані зберігаються локально і жодні дані не передаються, ризик несанкціонованого доступу мінімальний.

6. Зміни до цієї Політики конфіденційності

Ми можемо час від часу оновлювати нашу Політику конфіденційності. Ми повідомимо вас про будь-які зміни, розмістивши нову Політику конфіденційності на цій сторінці та оновивши дату "Останнє оновлення" вгорі цієї Політики конфіденційності.

7. Ваші права

Ви маєте право в будь-який час отримати доступ, оновити або видалити свою інформацію. Оскільки всі дані зберігаються локально, ви можете керувати цим через налаштування вашого браузера.

8. Зв'яжіться з нами

Якщо у вас виникли запитання щодо цієї Політики конфіденційності, будь ласка, зв'яжіться з нами за адресою zhidetskij@gmail.com.
